Query Quest — Round 1: MCQ Paper
================================

Instructions: 15 minutes. 20 multiple-choice questions. Each question 1 mark.

Questions:
1) Which condition is correct to check NULL values in SQL?
A) = NULL   B) == NULL   C) IS NULL   D) LIKE NULL
Answer: C

2) GROUP BY is mainly used to:
A) Filter rows   B) Sort records   C) Group rows for aggregate functions   D) Delete duplicates
Answer: C

3) Which SQL clause removes duplicate rows from query results?
A) UNIQUE  B) DISTINCT  C) FILTER  D) GROUP
Answer: B

4) Which aggregate function returns the number of rows?
A) SUM  B) COUNT  C) TOTAL  D) SIZE
Answer: B

5) Which join returns all rows from the left table and matched rows from the right table?
A) INNER JOIN  B) RIGHT JOIN  C) LEFT JOIN  D) FULL JOIN
Answer: C

6) Which statement changes existing rows in a table?
A) INSERT  B) UPDATE  C) ALTER  D) MODIFY
Answer: B

7) Which clause filters groups (post-aggregation)?
A) WHERE  B) HAVING  C) GROUP  D) FILTER
Answer: B

8) What does the PRIMARY KEY constraint ensure?
A) Non-null and unique values per row  B) Only numeric values  C) Faster queries  D) Foreign key enforcement
Answer: A

9) Which command removes a table and its data permanently?
A) DROP TABLE  B) DELETE TABLE  C) TRUNCATE TABLE  D) REMOVE TABLE
Answer: A

10) Which statement adds a new column to an existing table?
A) ADD COLUMN  B) ALTER TABLE ... ADD COLUMN  C) UPDATE TABLE  D) MODIFY TABLE
Answer: B

11) Which data type is best for storing variable-length text?
A) CHAR  B) VARCHAR  C) INT  D) DATE
Answer: B

12) Which index type enforces uniqueness on a column?
A) NORMAL INDEX  B) UNIQUE INDEX  C) FULLTEXT  D) HASH
Answer: B

13) Which SQL keyword is used to sort query results?
A) ORDER BY  B) SORT BY  C) GROUP BY  D) ARRANGE BY
Answer: A

14) Which clause is evaluated first in SQL query processing?
A) SELECT  B) WHERE  C) FROM  D) ORDER BY
Answer: C

15) Which statement is true about FOREIGN KEY?
A) It uniquely identifies a row in the same table  B) It creates referential integrity between tables  C) It stores binary data  D) It enforces non-null
Answer: B

Scoring & tie-breaker: Use accuracy first, then time taken. For ties, a short SQL practical question (5–10 minutes).
