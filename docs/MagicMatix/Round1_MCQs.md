MagicMatix — Round 1 (MCQ)
================================

Instructions: 30 minutes. Individual/team as per event. 20 MCQs.

1) Which number plays a vital role in circles?
A) 2.71  B) 3.14  C) 1.61  D) 0
Answer: B (Pi — 3.14)

2) Which of these is the golden ratio (approx)?
A) 1.41  B) 2.72  C) 1.61  D) 3.14
Answer: C

3) Which file format shares its abbreviation with a probability function?
A) PNG  B) PDF  C) DOCX  D) MP4
Answer: B (PDF — Probability Density Function / Portable Document Format)

4) The derivative of sin(x) is:
A) cos(x)  B) -sin(x)  C) tan(x)  D) 1
Answer: A

5) Which structure stores data in key-value pairs?
A) Array  B) Stack  C) Dictionary/Map  D) Queue
Answer: C

... (additional MCQs can be added)

Scoring: top teams advance to Round 2. Tie-breaker: least time, then both advance if tie remains.
