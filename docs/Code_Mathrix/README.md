Code Mathrix — Event Materials
===============================

Description:
Code Mathrix is a collaborative coding challenge testing logic, coordination, and problem-solving. Teams of 2 complete turn-based coding tasks under isolation rules.

Structure:
- Round 1: 30 minutes — logic MCQs and dry-run problems.
- Round 2: 40 minutes — live coding problems with turn-based rule (Player1 10m, Player2 10m, repeat).

Logistics:
- Systems with compilers/IDEs
- Printed question papers (≥25)
- Judges and volunteers (≥5)
